# BuildConnect Pilot

BuildConnect Pilot la ban chay thu website viec lam xay dung Da Nang.

## Chay local

```bash
npm start
```

Mo:

```text
http://127.0.0.1:4180/
```

## Deploy

Nen deploy bang Render Web Service.

```text
Environment: Node
Build Command: npm install
Start Command: npm start
```

Bien moi truong:

```text
NODE_ENV=production
DB_PATH=/var/data/db.json
```
